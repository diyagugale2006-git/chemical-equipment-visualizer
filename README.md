# Chemical Equipment Parameter Visualizer

Hybrid Web + Desktop Application for chemical equipment data visualization and analytics.

---

## Project Overview

This project is a hybrid application that works as:

- Web Application
- Desktop Application

Both applications use the same backend API.

Users can upload a CSV file containing chemical equipment parameters and visualize analytics such as:

- Equipment type distribution
- Average flowrate
- Average pressure
- Average temperature

---

## Tech Stack

### Backend
- Python Flask
- Pandas
- REST API

### Web Frontend
- HTML
- JavaScript
- Chart.js

### Desktop Application
- PyQt5
- Matplotlib

---

## Features

- CSV upload support
- Data analysis using Pandas
- Equipment type distribution chart
- Web visualization using Chart.js
- Desktop visualization using Matplotlib
- Common backend API
- Sample CSV file included
